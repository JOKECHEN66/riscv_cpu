------------------
testbench使用手册：
------------------

1. 	首先调用script.py脚本，命令行指令为python script.py -p [path] -s [suffix] -sp [save_path]
	p为获取目标测试文件的根目录；s为目标测试文件后缀；sp为导出列表文件的目录，通常无需修改
	参数的细节设置可以详见--help列表和具体代码

2.	生成目标文件后打开tb.v文件，修改第39行和49行
	注意测试指令和目标文件夹的一致性（I和M指令需保持一致）

3.	打开Modelsim仿真测试，tb文件设置的仿真时间上限为5ms
	指令批量测试的时间通常在1-2ms

4.	若需重新导入Modelsim中的工程文件，请在导入tb.v文件时参考setting.png中的设置将语法检验设置为SystemVerilog
	其余文件无特殊要求

5.	仿真过程中可以对波形分析和仿真结果截图，以用于后续文档编写
